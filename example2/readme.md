## 1 介绍

本例子给出链接脚本的例程。

- lma_vma.ld：设计输出段LMA属性是为了简单地创建一个ROM镜像。比如，下面的链接脚本创建三个输出段：一个名为“.text”，从0x1000开始；一个名为“.mdata”，它加载到“.text”段后面，其VMA是0x2000；一个名为“.bss”，在地址0x3000处保存未初始化的数据。符号_data定义为值0x2000,这个值保存VMA的位置寄存器，不是LMA值的。
