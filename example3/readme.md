本文介绍嵌入式存储的三种 存储方式的链接脚本写法。

- 存储方式A：表示4个输出段存放在RAM中
- 存储方式B：表示.text段和.rodata段存放在ROM中，.data段存放在RAM/ROM中，.bss存放在RAM中
- 存储方式C：表示.text段存放在ROM中，.rodata段存放在ROM2中，.data存放在RAM/ROM2中，.bss存放在RAM中
